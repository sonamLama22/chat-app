## **About The Project**

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

A simple react chat application.

### **Tech Stacks**

- React JS
- Mongo DB, Express
- Node JS
- React Router
- Redux
- Redux persist
- Material Design
- Ant Design & Material UI
- Formik & Yup for Form Validation in React
- Password encryption using Bcrypt
